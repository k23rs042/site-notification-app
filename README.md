# グッズまとめサイト（仮）

## 概要

アニメ・漫画などのグッズ情報をまとめて閲覧できるWebアプリです。

## 主なページ
- トップページ（グッズ一覧・タグ絞り込み）
- 作品一覧ページ
- ログインページ
- アカウント登録ページ

## URL
https://site-notification-app-1.onrender.com/

## セットアップ
```bash
npm run server
```
別のターミナルで以下を実行
```bash
cd site-notification-app
npm install
npm start
```

---

このアプリはReact + React Routerで構築されています。
